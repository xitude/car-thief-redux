# Car Theft 7 Redux

A top-down crime sim that runs entirely in your browser — no download, no signup, no install.

### ▶ [Play it here](https://xitude.github.io/car-thief-redux/)

You're working off a debt to a loan shark named Prince. To clear it you boost cars,
build and manage a crew, and decide what to do with the metal: fence it cheap, run it
through a chop shop, or let it cool and consign clean cars across cities for full retail.
Meanwhile the heat keeps climbing and rival crews start sniffing around your turf.

It autosaves, so you can dip in and out. There's real depth under the hood — crew
loyalty, debt thresholds, equipment slots — so read the encounter text; it matters.

## Repo layout

- `index.html` — the playable game (self-contained, what GitHub Pages serves)
- `react/` — the React source the game is built from
- `CarThief6*.html` — earlier versions, kept for history
